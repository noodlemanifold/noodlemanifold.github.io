Jump Flood Outlines

This is a fullscreen outline effect implemented in Unity's Universal Render Pipeline as a Scriptable Renderer Feature. It is based on this amazing article by Ben Golus. He explains how the Jump Flood algorithm can be used to generate a distance field to any seed pixels in real time. This is much more performant and allows for much wider outlines than standard edge detection approaches. I implemented the algorithm into the URP, and made some changes. I added weights to the distance field calculation to allow for outlines of varying widths, and made it possible to define internal outlines on objects by rendering them with a special texture that is completely decoupled from the object's normal texture. These changes give complete artist control to how the object outlines are detected and rendered. After many attempts, this is the best outline shader I have written by far, I am very proud of it!

Unity Graphing Library

Recently I wanted to make some 3D charts. I looked for a good free asset in Unity to do this, but I couldn't find any, so I just made my own! It can take in a csv file, parse it, and then display the results with many options to the user. Drag and Zoom are also both supported. Each data point is converted into a position within the chart's volume in a compute shader, and rendered as a 3D sphere with GPU instancing. I had over 20 charts, each with hundreds or thousands of points being rendered, and still had no performance issues. I am very proud of how this ended up coming out!
