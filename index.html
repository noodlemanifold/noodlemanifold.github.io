Rowland's Adventure

This was a much-to-ambitious 3D Platformer with a gravity gimmick for a 3 day game jam. Despite this, I still managed to finish a cute little demo before the deadline, and its fun! Soft Reminder

I needed a periodic, soft and silent reminder for therapy work, but I could not find any apps that allowed this, so I did it myself! This is a simple Android app that runs a custom vibration pattern at a user set rate. Unfortunately I cannot publish it on the Play Store because it needs to constantly set alarms to start foreground services in order to function. It works reliably and has helped me quite a bit! This project taught me Kotlin basics, the Android Process Lifecycle, and the Jetpack Compose UI Framework.

Wooden Croaks Bookstore

This was a group project with other developers I met in a discord. The idea was you ran a magical bookstore with gameplay similar to the game Unpacking. I coded the placement grid, holding and moving objects, a menu system, a dialogue system, a game sequencing system, and the audio system, while also doing the graphics programming for the grass & outline effects. I also made all the temporary props used in the demo. Unfortunately, my team members we unable to contribute much themselves, and the project eventually died. Honestly I'm still sad about it, I put in so much work and had so much passion for this game. At least I still learned a lot about Compute Shaders and Unity's new Universal Render Pipeline along the way, and I'm incredibly proud of what I was able to make for it.

3D Artwork Showcase

Modelling in 3D is skill I only recently gained proficiency in, so I don't have as many examples of my work, but I wanted to showcase some of my better projects here together!

Desert Bus 3D

I used to have a friend a while ago who loved the Desert Bus Charity Marathon, so I made a 3d version of the game for him! It has a fully procedural terrain system, with Perlin Noise, octaves, LOD and chunks. The textures are just triplanar mapped in a shader. This project taught me so much, it is where my skills in many advanced game development topics began.

Unity Graphing Library

Recently I wanted to make some 3D charts. I looked for a good free asset in Unity to do this, but I couldn't find any, so I just made my own! It can take in a csv file, parse it, and then display the results with many options to the user. Drag and Zoom are also both supported. Each data point is converted into a position within the chart's volume in a compute shader, and rendered as a 3D sphere with GPU instancing. I had over 20 charts, each with hundreds or thousands of points being rendered, and still had no performance issues. This was a niche one but I still think its cool.

3D Game of Life

This was one of my very first Unity projects! It allows you to build and save any initial conditions for a simulation you like. You can also change the rules for the simulation along the top (Green for create, Yellow for Survive, Red for Death) for any number of neighbors. In addition to teaching me the basics of Unity, I also learned how to serialize data, write it to disk, and read it back later.
